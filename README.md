# AI Word Visualizer for Dyslexic Learners

An AI-powered accessibility tool designed to help dyslexic learners improve reading fluency through color-coded syllable visualization, audio pronunciation, and multisensory learning techniques.

---

## Problem Statement

Dyslexia affects the way individuals process written language, making reading, spelling, and word recognition challenging. Many existing learning tools focus on a single learning method and fail to provide comprehensive support.

This project addresses these challenges by combining:

- Visual Learning
- Auditory Learning
- Kinesthetic Learning

into one interactive platform that helps learners understand and pronounce words more effectively.

---

## Objectives

- Automatically break words into syllables
- Color-code syllables for better visual recognition
- Generate audio pronunciation for phonetic understanding
- Support interactive tracing activities
- Improve reading fluency and confidence in dyslexic learners

---

## Features

### Visual Learning Support
- Automatic syllable segmentation
- Unique color assignment for each syllable
- High-contrast and learner-friendly interface

### Audio Assistance
- Text-to-Speech pronunciation
- Phonetic reinforcement through audio playback

### Interactive Learning
- Tracing mode for kinesthetic learning
- Practice activities for better retention

### Accessibility Focus
- Dyslexia-friendly design
- Easy-to-use interface
- Multisensory learning approach
